## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/spon7ge/Track-Your-Money.git
cd Track-Your-Money
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env` file:

```bash
cp .env.example .env
```

Then edit `.env` with your Firebase configuration.

4. Start the development server:

```bash
npm start
```
